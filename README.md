# Smart Waste Segregation System (TrashDeal)

## Overview
This project was created as part of a Design Thinking course. The goal is to encourage people to dispose of waste properly using a reward-based system.

## Problem
Many people mix wet and dry waste, which makes recycling difficult and increases pollution.

## Solution
A mobile application prototype called TrashDeal that rewards users for properly segregating waste.

## Features
- User login system
- Waste collection vehicle tracking
- Reward points for waste disposal
- Points redemption for groceries

## Prototype
The application prototype was designed using Figma.

## Goal
Encourage sustainable waste disposal behavior and improve recycling efficiency.
